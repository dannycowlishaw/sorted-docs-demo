• Allocate Shipment with Quote Endpoint 
    PUT https://api.sorted.com/pro/shipments/{reference}/allocate/quote/{quote_ref}

    To allocate an individual shipment based on a specific delivery quote from a carrier, use the Allocate Shipment with Quote endpoint.

    The Allocate Shipment with Quote endpoint takes the {reference} of the shipment you want to allocate and the {quote_ref} of a particular quote for that shipment. Once the request is received PRO attempts to allocate the shipment to the carrier service specified in the quote, and returns an Allocate Result.

    Note

    For full reference information on the Allocate Shipment with Quote endpoint, see the Allocate Shipment with Quote page of the API reference.
