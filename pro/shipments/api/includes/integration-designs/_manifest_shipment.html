• Manifest Shipments Endpoint 
    PUT https://api.sorted.com/pro/shipments/{reference}/manifest

    Once you've created a shipment, allocated it to a carrier service and printed labels for it, you're ready to manifest it. To manifest a shipment, use the Manifest Shipment endpoint. In the context of SortedPRO, the term "manifesting" refers to collating, formatting and transmitting the shipment data to carriers.

    The Manifest Shipment endpoint enables you to manifest an individual shipment. Once PRO has received the request, it attempts to manifest the specified shipment and returns a manifest_result containing a unique reference for the manifest item and details of the carrier and service that the shipment was manifested with.

    Caution

    Every successful request to a manifest endpoint results in data being transmitted to the carrier. Therefore, Sorted strongly advise that manifesting of scheduled shipments is aligned with the carrier collection times from the warehouse.

    Note
    • For full reference information on the Manifest Shipment endpoint, see the Manifest Shipment page of the PRO v2 API Reference.
    • For a user guide on manifesting shipments, see the Manifesting Shipments page.
